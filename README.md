# E-Commerce-App
Project Overview

This project is a console-based e-commerce shopping cart system developed using Java.
It demonstrates core programming concepts such as classes, objects, collections, loops, and conditional logic.

The application allows users to view available products, add items to a cart, update quantities, remove products, and generate a bill with tax calculation.
-------------------------------------------------------------------------------------------------
Features
-------------------------------------------------------------------------------------------------
1.Display available products with price and stock

2.Add items to the shopping cart

3.Remove items from the cart

4.Increase cart item quantity

5.Decrease cart item quantity

6.Prevent adding items beyond available stock

7.Calculate subtotal

8.Apply 5 percent tax

9.Display final bill amount

Menu-driven console interface

Clean and readable output format

-------Tech Stack-------------------------------------------------------------------------------

Language: Java

Application Type: Console Based

IDE: VS Code / IntelliJ / Eclipse

Java Version: Java 8 or above
